# Soluna

![Banner](https://raw.githubusercontent.com/EndOverdosing/Soluna-API/refs/heads/main/images/banner.png)

A Roblox script loader supporting 20+ games with a clean interface, persistent settings, and auto-load functionality.

---

## Getting Started

Execute the following in your preferred executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/EndOverdosing/Soluna-API/main/main.lua", true))()
```
